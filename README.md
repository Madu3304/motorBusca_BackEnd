# Descrição: 
Aplicação web para localização e filtragem de termos específicos em documentos TXT e PDF.

#Guia de Execução:

1.0 - cd motorBusca_BackEnd
Ativar venv:
Windows: .\venv\Scripts\activate
Instalar dependências: pip install fastapi uvicorn python-multipart PyPDF2

Rodar API: python -m uvicorn main:app --reload
